# Idara Al-Khair Welfare Society

Official Digital Ecosystem for Idara Al-Khair, built with Next.js, TypeScript, and Bootstrap.

## Features
- Professional Component-Based Architecture
- 3D Card Hover Effects for Projects
- Modern Responsive Navbar and Footer
- Developed with Next.js App Router

## Getting Started
Run the development server:
```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
